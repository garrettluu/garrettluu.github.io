R3 - A 3D grapher for the TI84PCE

Github link:

https://github.com/garrettluu/r-cubed

A 3D grapher that takes in equations in reverse-polish (postfix) notation and outputs a graph of the function. Written in C using the C toolchain. Deprecated, soon to be rewritten using Cuboid.

Cuboid - A 3D graphics library for the TI84PCE

Github link:

https://github.com/garrettluu/cuboid

A 3D graphing library based on vectors, supports writing custom projections. Planning support for linear transformations, infix parsing, and higher-dimension math. Written in C using the C toolchain.
